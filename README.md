# Binti Project — AfriCraft NGO

Eine Spenden- und Info-Website für das **Binti-Projekt** von **AfriCraft NGO** (African Handcraft Organization, Nr. 00NGO/1307) in Tansania. Das Projekt versorgt Schulmädchen mit wiederverwendbaren Menstruations-Hygiene-Sets und Aufklärung, damit sie ihre Periode mit Würde managen und in der Schule bleiben.

## Über die Seite

- **Eine einzige Datei** (`index.html`) — alle Bilder, Schriften-Verweise, Stile und Skripte sind enthalten. Kein Build, keine Abhängigkeiten.
- **Dreisprachig**: Deutsch (Standard), Englisch, Kiswahili — umschaltbar oben rechts.
- Interaktiver „Binti-Set zusammenstellen"-Bereich, Wirkungs-Zahlen, FAQ, Impressum/Datenschutz/AGB, GoFundMe-Spendenlink.
- Afrikanisch inspiriertes Design (Kanga-Farben & -Muster).

## Ansehen

Einfach `index.html` im Browser öffnen (Doppelklick).

## Kostenlos veröffentlichen mit GitHub Pages

1. Dieses Repository auf GitHub hochladen (siehe unten).
2. Im Repo: **Settings → Pages**.
3. Unter **Branch** `main` und Ordner `/ (root)` wählen, **Save**.
4. Nach ein bis zwei Minuten ist die Seite öffentlich erreichbar unter
   `https://<dein-github-name>.github.io/binti-project/` — ohne Passwort, für alle sichtbar.
